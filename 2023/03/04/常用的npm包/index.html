常用的npm包

发表于 2023-03-04 更新于 2023-04-27 分类于 常用的npm包

ms

ms 是一个毫秒转换实用程序，可以轻松地将各种时间格式转换为毫秒。

p-limit

p-limit 用于有限的并发运行多个 promise &async 函数。

Ajv

Ajv 是一个适用于 Node.is 和浏览器的最快 JSON 验证器。它支持 JSON Schema Draft-04/06/0712019-09/2020-12 和JSON 类型定义(RFC8927)。

uuid

uuid 用于在 JavaScript 中生成符合 RFC4122的 UUID。其支持生成 RFC4122 1、3、4 和5版本的UUID，支持跨平台使用，通过了uuid命令行实用程序。

qs

一个简单易用的字符串解析和格式化库

mitt

微型 200b 功能事件发射器/发布订阅.

lodash lodash-es

是一个用于 JavaScript 的实用程序带库,通过消除处理数组、数字、对象、字符串等的麻烦，使 JavaScript 变得更容易

dayjs momentjs

它为现代浏览器解析、验证、操作和显示日期和时间

validator

Validator是一个字符串验证器和清理器库

await-to-js

摆脱 async、await 中繁琐的 try、catch
