发表于 2023-05-05 更新于 2023-05-04 分类于 react

11 个 React 错误用法

  • 渲染列表时，不使用 key
  • 直接通过赋值方式修改 state 值
  • 将 state 值直接绑定到 input 标签的 value 属性
  • 执行 setState 后直接使用 state
  • 使用 useState + useEffect 时出现无限循环
  • 忘记在 useEffect 中清理副作用
  • 错误的使用布尔运算符
  • 没有定义组件参数类型
  • 把字符串当做数值传递到组件
  • 没有以大写字母开头的组件名称
  • 错误的为元素绑定事件
